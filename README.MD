# Creator Platform Backend

## Technologies

- Node.js
- Firebase Admin SDK
- Firebase Cloud Functions
- Google Cloud

### Getting started

**Install dependencies**

```
npm i
```

**Development**

```
npm run dev
```

**Deployment**
Note: Must have Google Cloud CLI installed. Follow the steps here to get it setup https://cloud.google.com/sdk/docs/install

```
npm run deploy
```
